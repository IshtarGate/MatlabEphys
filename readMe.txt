This document describes the most important m-files that the normal 
	person will use

jahCurrentClampAnalysis.m

    This file analyzes current clamp data

jahGetABFfileNames.m
    This file can get all the .abf file's names to make an excel book
    for jahCurrentClampAnalysis.m.


jahAbfToPrism2.m

    This file quickly convert large abf files into a smaller dataset 
    for use in prism
